# getRadWeather calculates radiation from DWD weather data

getRadWeather calculates radiation from DWD weather data

## Usage

``` r
getRadWeather(WeatherTab)
```

## Arguments

- WeatherTab:

  Data frame with DWD data including sunshine duration without radiation

## Value

Data frame with radiation in \[MJ.m-2.d-1\] and the other columns

## Details

also saturation deficit is calculated
