# transWindSpeed corrects the wind speed for the required height considering the measurement height

transWindSpeed corrects the wind speed for the required height
considering the measurement height

## Usage

``` r
transWindSpeed(inWind, inHeight, outHeight, vegi.height)
```

## Arguments

- inWind:

  original wind speed data \[m/s\]

- inHeight:

  measurement height of original wind speed data \[m\]

- outHeight:

  reference height to be wind speed should be converted \[m\]

- vegi.height:

  vegetation height \[m\]

## Value

corrected wind speed data \[m/s\]
