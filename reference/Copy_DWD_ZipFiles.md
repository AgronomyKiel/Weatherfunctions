# Copy_DWD_ZipFiles makes a local copy of DWD weather data as zip files

Copy_DWD_ZipFiles makes a local copy of DWD weather data as zip files

## Usage

``` r
Copy_DWD_ZipFiles(DWD_ftp_, LocalCopy_DWD_ftp_)
```

## Arguments

- DWD_ftp\_:

  ftp address of DWD weather data

- LocalCopy_DWD_ftp\_:

  directory for storing local copy of DWD data

## Value

Nothing
