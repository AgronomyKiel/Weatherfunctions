## Usage

``` r
setplotbackground(plot, BaseSize = 18)
```

## Arguments

- plot:

  a plot to be customized

- BaseSize:

  the base font size

## Value

a plot with customized design and an adjusted font size

## Examples

``` r
 setplotbackground(p, BaseSize=18)
```
