# Alternative function for correction of windspeed to reference height

Alternative function for correction of windspeed to reference height

## Usage

``` r
windheight(ui, zi, zo)
```

## Arguments

- ui:

  original windspeed \[m/s\]\]

- zi:

  measurement height \[m\]

- zo:

  reference height \[m\]

## Value

uo corrected windspeed \[m/s\]
